德森特公司环境配置

  • Platform: gitlab

  • 配置要求

    • Node JS: v14.15.0
    • MongoDB: v4.4.0->v5.0.5
    • MySQL: v5.5.7
    • Git: v2.24.3
    • Maven: v3.6
    • Nginx: v1.19.4

  • CLOUDOPS

    • git clone 拉取代码
    • npm insatll 安装依赖
    • npm start 启动程序

  • APPLICATION

    • npm install lerna -g安装lerna
    • 执行lerna bootstrap
    • 执行lerna exec -- node ../../tools/make_debug_env.js
    • Debug启动程序
