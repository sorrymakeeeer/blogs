基于HEXO框架个人博客搭建

背景

介绍基于HEXO静态博客框架搭建个人博客

工具：Windows10 laptop

前期准备

  1. 安装nodeJS（并配置npm repository地址，配置淘宝仓库地址即可。cnpm可以不装）
  2. 安装并配置git（参考百度）
  3. 安装NotePad++/VSCode/Typora文本编辑器（习惯命令行操作可以忽略该步骤）

安装HEXO

  1. npm i hexo -g

  2. 配置HEXO的环境变量

  1. 命令行执行 hexo -v 正确显示版本信息表示安装成功

运行

  1. 任意位置创建文件夹作为博客目录，命令行执行hexo init，初始化服务
  2. 执行npm i，安装依赖
  3. 执行hexo g，编译程序
  4. 执行hexo server/hexo s启动服务
  5. 浏览器访问localhost:4000即可

部署

参考网络上的文章，基于码云)/GitHub部署

  1. 注册登录码云账号

  2. 创建远程仓库

  3. npm i 安装hexo git部署插件 hexo-deployer-git

  4. 在创建的博客文件夹下，打开_.config.yml配置文件

  5. hexo clean hexo g hexo deploy 重新生成并且部署

  6. 远程仓库 service 开启
